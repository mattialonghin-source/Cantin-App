ℹ️ Informazioni

Senza API key: L'app funziona completamente offline con un database locale di abbinamenti per oltre 50 denominazioni e vitigni italiani.

Con API key: Gli abbinamenti vengono generati da Claude AI in modo personalizzato in base ai dettagli del tuo vino specifico. Ideale per abbinamenti più precisi.

Privacy: Tutti i dati sono salvati localmente sul dispositivo (IndexedDB). La API key è salvata solo nel localStorage del browser e non viene mai trasmessa se non direttamente ad Anthropic.
